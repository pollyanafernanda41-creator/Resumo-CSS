Home

O conceito deste portfólio foi inspirado em jogos de plataforma clássicos para explorar recursos de HTML, CSS e JavaScript de forma criativa e interativa. A escolha dessa estética não teve como objetivo criar um ambiente infantil, mas sim desenvolver uma experiência capaz de despertar curiosidade, atenção e familiaridade durante a navegação. Cada elemento foi pensado considerando a experiência do usuário, desde a organização do layout até as animações, interações e feedbacks visuais. Os componentes interativos presentes no projeto foram utilizados como forma de praticar conceitos de desenvolvimento front-end, acessibilidade, responsividade e manipulação do DOM com JavaScript, transformando o portfólio em um ambiente que demonstra não apenas informações sobre mim, mas também minhas habilidades técnicas na construção de interfaces. Mais do que apresentar meu currículo, este projeto busca demonstrar como utilizo a tecnologia para criar experiências envolventes, intuitivas e visualmente organizadas.
